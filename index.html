FUSE: Fantasy User Stat Enhancer

FUSE lets you enhance player stats in your fantasy football app by layering in your own custom data to simplify roster selection, free agent scouting, and trade evaluations.

This is currently in beta. Submit bugs, issues, or feature requests to the github repo.

Install

  1. Drag the link below to your bookmarks bar
  2. Click the bookmarklet to run FUSE on a page
